# Rice Management App

## Project Overview

This is a simple rice trading management web app for a family business in Vietnam.

The system manages:
- Rice purchasing from farmers
- Brokers and commission debts
- Transport trips and transport debts
- Drying and milling processing
- Processing debts
- Supporting documents and attachments
- PDF/Excel exports
- Loss tracking during transport/drying/milling

The app should prioritize:
- Simplicity
- Fast data entry
- Easy mobile usage
- Responsive web design
- Easy PDF export
- Clear debt tracking
- Minimal clicks

Do NOT over-engineer the system.

Avoid:
- Microservices
- Complex DDD architecture
- Enterprise ERP patterns
- Premature optimization
